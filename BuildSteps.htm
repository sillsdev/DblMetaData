DblMetaData Build steps

  1. Edit source
  2. Right click Test Project and Rebuild all
  3. Edit UpdateMetaData-1.2.xsl and rerun visual studio tests
  4. Copy content of UpdateMetaData-1.2.xsl into ApplyUpd12.py
  5. from XP virtual machine, python setup.py to rebuild ApplyUpd12
  6. copy ApplyUpd12.exe into dist folder
  7. Update version number in setup.py
  8. Update version # in DblMetaData project on Application and Publish tabs
  9. Update version # in Setup1.vdproj (when Visual Studio closed)
  10. Right click DblDataPrepSetup project and rebuild all
  11. Rename Setup1\Debug\DblDataPrepSetup to include version #
  12. Right click DblMetaData folder and HG Commit new and changed files.
  13. Use HG workbench on same folder. Right click revision and export to zip archive.
  14. Copy created msi file and source to ETEN-DBL drop box
